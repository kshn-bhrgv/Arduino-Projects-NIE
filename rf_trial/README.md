# Four-LED Alternating Pattern

This sketch drives four LEDs on pins 3–6 in an alternating pattern: pins 3, 5, and 6 go HIGH while pin 4 goes LOW, then after 300 ms the pattern inverts. This is a simple LED pattern test with no sensor input.

## Hardware

- Arduino Uno
- 4x LEDs (with current-limiting resistors)

## Pin Connections

| Component | Pin |
|-----------|-----|
| LED A | 3 |
| LED B | 4 |
| LED C | 5 |
| LED D | 6 |
