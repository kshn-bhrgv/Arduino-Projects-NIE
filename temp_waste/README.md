# Android Bluetooth RGB LED Parser (Code Stub)

This file contains only commented-out code that was intended to parse RGB colour and LED number data received from an Android app over Bluetooth (SoftwareSerial). The code extracts hex R, G, B values and an LED number from a formatted string. There is no active code; the file is a reference or work-in-progress stub.

## Hardware

- Arduino Uno
- Bluetooth module (HC-05 or similar)
