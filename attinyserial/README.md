# attinyserial

ATtiny sketch that reads the analog value from pin A1 and transmits it over SoftwareSerial (pins 3/4) every 50 ms. Useful for wirelessly relaying sensor data from an ATtiny to a host.

## Hardware

- **Platform:** ATtiny (requires ATtinyCore board support)
- **Pins:** Analog input on A1; SoftwareSerial RX=3, TX=4
