# _7447

Drives four digital output pins to cycle through BCD values 0-4 with 500 ms delays between each. Intended for use with a 7447 BCD-to-7-segment decoder IC.

## Hardware

- **Platform:** Arduino Uno (ATmega328P)
- **Pins:** A=8, B=7, C=10, D=4 (BCD inputs to 7447)
- **IC:** 74LS47 / CD4511 BCD-to-7-segment decoder
