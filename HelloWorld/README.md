# HelloWorld

Displays "hello" and "world" on an HD44780-compatible LCD using the LiquidCrystal library. Configured for a 16x1 display addressed as 8x2.

## Hardware

- **Platform:** Arduino Uno (ATmega328P)
- **Pins:** RS=12, Enable=11, RW=10, D4=5, D5=4, D6=3, D7=2
- **Display:** 16x1 (or 8x2) LCD

## Usage

1. Wire the LCD to the pins above with a 10K contrast potentiometer on V0.
2. Upload and the display will show "hello" / "world".
