# Three LED Sequential Blink

This sketch blinks three LEDs on pins 8, 9, and 13 in sequence. Each LED turns on for 1 second then off for 0.5 seconds before the next one lights. It is a basic multi-LED blink test.

## Hardware

- Arduino Uno
- 3x LEDs (with current-limiting resistors)

## Pin Connections

| Component | Pin |
|-----------|-----|
| LED A | 8 |
| LED B | 9 |
| LED C | 13 |
