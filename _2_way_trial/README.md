# _2_way_trial

IR receiver sketch that drives pin 7 HIGH while a specific IR code is received and LOW otherwise. Demonstrates basic one-button IR remote control of a single output.

## Hardware

- **Platform:** Arduino Uno (ATmega328P)
- **Pins:** IR receiver on pin 11, output on pin 7
- **Libraries:** IRremote
