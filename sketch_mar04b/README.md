# LCD Hello Sketch

This sketch initialises a 16x2 LCD using the LiquidCrystal library and prints " hi**" on the first row at startup. The loop is empty so the text remains static. It is a minimal LCD hello-world test.

## Hardware

- Arduino Uno
- 16x2 character LCD

## Pin Connections

| LCD Pin | Arduino Pin |
|---------|-------------|
| RS | 12 |
| EN | 11 |
| D4 | 5 |
| D5 | 4 |
| D6 | 3 |
| D7 | 2 |
