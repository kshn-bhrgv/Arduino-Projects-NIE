# analogcommosci

Measures the time period between consecutive HIGH pulses on digital pin 8 and prints the result to Serial. Intended for measuring the period of a digital signal (e.g. from a TSOP or oscillator).

> Note: The sketch has a variable naming bug (`oldtp`/`newtp`) that prevents compilation as-is.

## Hardware

- **Platform:** Arduino Uno (ATmega328P)
- **Pins:** Digital input on pin 8
