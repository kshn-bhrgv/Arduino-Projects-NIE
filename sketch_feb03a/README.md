# Joystick Module Reader

This sketch reads X-axis and Y-axis analog values and a digital switch press from a joystick module, printing all three readings to the Serial Monitor at 115200 baud every 100 ms. It is a basic test sketch for verifying joystick wiring and reading ranges.

## Hardware

- Arduino Uno
- Joystick module (with X, Y analog outputs and a push-button switch)

## Pin Connections

| Component | Pin |
|-----------|-----|
| Joystick Switch | 2 |
| Joystick X-axis | A0 |
| Joystick Y-axis | A1 |
