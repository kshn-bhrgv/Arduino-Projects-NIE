# Serial LED Toggle

This sketch reads single characters from the Serial Monitor and toggles the built-in LED on pin 13. Sending 'O' turns the LED on; sending 'N' turns it off. The received character is echoed back to the Serial Monitor. It is a minimal test to verify serial communication and digital output.

## Hardware

- Arduino Uno

## Pin Connections

| Component | Pin |
|-----------|-----|
| LED (built-in) | 13 |
