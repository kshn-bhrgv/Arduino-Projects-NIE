# Pin 13 Blink

This is a minimal blink sketch that toggles the built-in LED on pin 13 HIGH and LOW with a 500 ms delay between states. It is used to verify basic Arduino functionality and upload capability.

## Hardware

- Arduino Uno

## Pin Connections

| Component | Pin |
|-----------|-----|
| LED (built-in) | 13 |
