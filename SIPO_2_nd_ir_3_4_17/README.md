# SIPO_2_nd_ir_3_4_17

Drives a 74HC595 shift register via an IR remote with an updated pin mapping (DATA=2, LED=7). Continuously sets both DATA and LED pins HIGH — appears to be a minimal output-state test for the shift register data line.

## Hardware

- **Platform:** Arduino Uno (ATmega328P)
- **Pins:** DATA=2, LED=7

## Usage

Upload and observe that pins 2 and 7 are driven HIGH continuously.
