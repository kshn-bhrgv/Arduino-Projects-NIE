# Analog Max Value Logger

This sketch continuously reads the analog value from pin A0 and prints it to the Serial Monitor at 9600 baud. Commented-out code also tracked and reported when a new maximum value was encountered. It is a basic diagnostic tool for calibrating or characterising an analog sensor.

## Hardware

- Arduino Uno
- Analog sensor or potentiometer

## Pin Connections

| Component | Pin |
|-----------|-----|
| Analog Sensor | A0 |
