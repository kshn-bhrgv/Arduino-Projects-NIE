# LED Blink with Switch

This sketch controls two LEDs on pins 11 and 5 based on the state of a switch connected to pin 4. When the switch is HIGH both LEDs blink alternately with 500 ms delays; otherwise only the second LED (pin 5) stays on. It is a simple conditional blink experiment.

## Hardware

- Arduino Uno
- 2x LEDs (with current-limiting resistors)
- Push button or switch

## Pin Connections

| Component | Pin |
|-----------|-----|
| LED A | 11 |
| LED B | 5 |
| Switch | 4 |
