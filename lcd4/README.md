# LCD Autoscroll Text Display

This sketch initialises a 16x2 LCD and uses the `autoscroll()` feature to continuously scroll the text "Shata Th" on row 0 and "ari!" on row 1. The display is updated every 500 ms, causing the text to shift horizontally with each iteration.

## Hardware

- Arduino Uno
- 16x2 character LCD

## Pin Connections

| LCD Pin | Arduino Pin |
|---------|-------------|
| RS | 12 |
| RW | 11 |
| Enable | 10 |
| D4 | 5 |
| D5 | 4 |
| D6 | 3 |
| D7 | 2 |
