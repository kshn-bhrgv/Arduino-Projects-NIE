# LCD Static Text Display

This sketch initialises a 16x2 LCD using the LiquidCrystal library and prints the static text "shatatheri" on the first row at startup. The loop does nothing; the text remains on screen indefinitely.

## Hardware

- Arduino Uno
- 16x2 character LCD

## Pin Connections

| LCD Pin | Arduino Pin |
|---------|-------------|
| RS | 12 |
| RW | 11 |
| Enable | 10 |
| D4 | 5 |
| D5 | 4 |
| D6 | 3 |
| D7 | 2 |
