# Digital Pin Output Test

This sketch sets pin 7 HIGH and pin 13 LOW in the loop. Pin 11 is also set up as an output. It is a minimal wiring or logic-level verification sketch with no dynamic behaviour.

## Hardware

- Arduino Uno
- LEDs or logic probes for verification

## Pin Connections

| Component | Pin |
|-----------|-----|
| Output A | 11 |
| Output B (HIGH) | 7 |
| Output C (LOW) | 13 |
